安装

全局安装 vue-cli

npm install –g @vue/cli

安装webpack

npm install -g webpack

创建项目

  • 图形化创建一个项目
    vue ui //package管理 设置为npm
  • 创建一个基于webpack模板的新项目
    vue init webpack my-project //出错则先运行 npm install -g @vue/cli-init
    • rooter yes 其他的都 no

安装依赖包

cd my-project
npm install //安装依赖包
npm run serve //ui让项目跑起来
npm run dev //webpack让项目跑起来
